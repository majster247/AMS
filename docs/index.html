About AMS-OS

AMS-OS is a simple and lightweight operating system designed for enthusiasts and developers. Its not like another Unix based shit - its full writed by hand and compiler with own compilator in own language

Key Features

  • Real time kernel
  • Own filesystem called "ShadowWizard" and bootloader called "Baobab"
  • Unique in a world of nigger tech
  • 16/32/64 bit port (fuck all ARM niggers)
  • Network & VGA driver (up to 8k 30.0Hz support in pre alpha, HDMI drivers in future)
